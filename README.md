# Ozed Hub Tech

**Ozed Hub Tech** is a blockchain service provider, focused on building secure, transparent, and interoperable decentralized infrastructure.

Our flagship project — **Ozed Network** — is a **Layer-1 blockchain** designed for **data sovereignty**, **cross-chain collaboration**, and **governance transparency**.

---

## Current Focus — Ozed Network (MVC Phase)

The **Minimum Viable Chain (MVC)** focuses on the foundation of Ozed Network:

- **Core Runtime** — Substrate-based runtime and kernel interface (OKI)  
- **Data Layer** — IPFS integration with incentivized pinning  
- **Node Communication** — P2P discovery and light-client connectivity  
- **Governance Telemetry** — On-chain voting, proposals, and metrics
