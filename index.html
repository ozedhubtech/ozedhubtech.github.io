About Ozed Hub Tech

Ozed Hub Tech is a blockchain technology studio. We focus on building foundational infrastructure for decentralized data interoperability, hybrid (public/private) behaviors, and strong governance models. Our approach blends Substrate-based runtimes, IPFS-based data layers, and telemetry-driven governance.

Why we exist

Data sovereignty and accessible infrastructure are core to our mission. By creating tools that keep control of data with individuals and organizations, we aim to unlock new forms of collaboration and secure data exchange.

Roadmap — Milestone 1 (MVC)

  • Core Runtime — Substrate runtime and kernel interface (OKI)
  • Data Layer — IPFS integration and incentivized pinning
  • Node Comm — P2P node discovery and light client support
  • Governance — Telemetry, on‑chain voting, and proposals
